## `assess_returns()`


```r
returns <- Xday_returns(data, Xday=3, log.returns = T)

assess_returns(returns, "mean")
```

```
##   Rank  Names  Value
## 1    1 Fund_3 -0.17%
## 2    2 Fund_1 -0.52%
## 3    3 Fund_2 -0.57%
```

```r
assess_returns(returns, "sd")
```

```
##   Rank  Names  Value
## 1    1 Fund_1  6.19%
## 2    2 Fund_2 12.63%
## 3    3 Fund_3 14.95%
```

```r
assess_returns(returns, "S.R")
```
