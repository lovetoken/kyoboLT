## `assess_returns()`


```r
returns <- Xday_returns(data, Xday=3, log.returns = T)

assess_returns(returns, "mean")
```

```
##   Rank  Names  Value
## 1    1 Fund_3 -0.74%
## 2    2 Fund_2 -0.74%
## 3    3 Fund_1 -1.37%
```

```r
assess_returns(returns, "sd")
```

```
##   Rank  Names  Value
## 1    1 Fund_1  7.65%
## 2    2 Fund_3  8.68%
## 3    3 Fund_2 10.39%
```

```r
assess_returns(returns, "S.R")
```
