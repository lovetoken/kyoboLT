Assess funds using returns xts dataset

Usage

assess_returns(returns.xts, method = c("mean", "sd", "S.R"), ...)

Arguments

returns.xts
an xts object about returns dataset
method
returns mean or sd or Sharpe ratio

Value

data.frame type about assessment result

Description

Assess funds using returns xts dataset

Examples

returns <- sample_index %>% xdiff_returns(1)

assess_returns(returns, "mean")


  Rank  Names Value
1    1 Fund_2 1.31%
2    2 Fund_1 0.97%
3    3 Fund_3 0.39%



assess_returns(returns, "sd")

assess_returns(returns, "S.R")
