## `assess_returns()`


```r
returns <- Xday_returns(data, Xday=3, log.returns = T)

assess_returns(returns, "mean")
```

```
##   Rank  Names   mean
## 1    1 Fund_2  0.47%
## 2    2 Fund_3  0.11%
## 3    3 Fund_1 -1.01%
```

```r
assess_returns(returns, "sd")
```

```
##   Rank  Names     sd
## 1    1 Fund_3  5.59%
## 2    2 Fund_1  9.95%
## 3    3 Fund_2 10.28%
```

```r
assess_returns(returns, "S.R")
```

```
##   Rank  Names     S.R
## 1    1 Fund_2   4.53%
```
