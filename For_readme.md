## `assess_returns()`


```r
returns <- Xday_returns(data, Xday=3, log.returns = T)

assess_returns(returns, "mean")
```

```
##   Rank  Names  Value
## 1    1 Fund_1  0.27%
## 2    2 Fund_3 -0.10%
## 3    3 Fund_2 -0.60%
```

```r
assess_returns(returns, "sd")
```

```
##   Rank  Names Value
## 1    1 Fund_2 6.34%
## 2    2 Fund_3 9.15%
## 3    3 Fund_1 9.41%
```

```r
assess_returns(returns, "S.R")
```
